# 2.举例说明：

**比对序列为：A = TGTTACGG，B = GGTTGACTA**

## 2.1确定置换矩阵和空位罚分办法

```
置换矩阵s(ai,bj)= {+3，ai==bj }
　　   　　　　　　{-3，ai!=bj }
```
即碱基匹配时分数+3，不匹配时分数-3

    空位罚分Wk=(k-1)+2

即第一个空位得分-2，随后得分递增减１，即连续两个空位得分-3，连续三个空位得分-4...

## 2.2创建矩阵并初始化

需要创建的矩阵有H(得分矩阵)，E(B序列空位延伸罚分矩阵)，F(A序列空位延伸罚分矩阵)。

在初始化时一般会在H矩阵的最左上方空位赋一个初值(8)，右侧值和下侧值为拓展空位罚分的数值。
